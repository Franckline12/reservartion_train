# Reservation Train

Application web pour la réservation de trains.

## Installation

1. Cloner le dépôt :
```bash
git clone https://github.com/Franckline12/reservartion_train.git
2.Installer le backend :
cd backend
npm install
3. Installer le frontend :
cd ../frontend
npm install
Lancement

Backend : npm run dev (ou node server.js)

Frontend : npm run dev (si Vite)

Fonctionnalités

Gestion des trains et trajets

Réservation de billets

Tableau de bord admin
